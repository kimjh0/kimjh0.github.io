MSVC 1800(Visual Studio 2013) 이상부터 stdbool.h, inttypes.h 지원 [^1]

MSVC 1600(Visual Studio 2010) 이상부터 stdint.h 지원 [^2]

그 미만은 아래를 참고하여 직접선언하여 사용

inttypes.h, stdint.h 는 각주 3번에서 가져옴 (revision 26) [^3]

## stdbool.h
- 정의
```c
#define bool int
#define true TRUE
#define false FALSE
```
- 사용
```c
#if _MSC_VER >= 1800
  #include <stdbool.h>
#else
  #include "msvc_stdbool.h"
#endif
```

#define imaxabs _abs64

// 7.8.2.2 The imaxdiv function

// This is modified version of div() function from Microsoft's div.c found
// in %MSVC.NET%\crt\src\div.c
#ifdef STATIC_IMAXDIV // [
static
#else // STATIC_IMAXDIV ][
_inline
#endif // STATIC_IMAXDIV ]
imaxdiv_t __cdecl imaxdiv(intmax_t numer, intmax_t denom)
{
   imaxdiv_t result;

   result.quot = numer / denom;
   result.rem = numer % denom;

   if (numer < 0 && result.rem > 0) {
      // did division wrong; must fix up
      ++result.quot;
      result.rem -= denom;
   }

   return result;
}
