MSVC 1800(Visual Studio 2013) 이상부터 stdbool.h, inttypes.h 지원한다. [^1]  
그 미만은 직접 선언해야 함

## stdbool.h
```c++
#if _MSC_VER >= 1800
  #include <stdbool.h>
#else
  #define bool int
  #define true TRUE
  #define false FALSE
#endif
```
